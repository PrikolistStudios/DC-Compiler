Подробное описание

Класс транслятора

Класс, отвечающий за синтаксический и семантический анализ, а также генерацию кода (ПОЛИЗ) для интерпретатора. Принимает на вход два файла - последовательность лексем ("separated.txt") и список операций с указанием приоритета ("operations.txt") Все функции без описания используются для анализа соответствующих элементов языка из грамматики
