\doxysubsection{Подробное описание}
Класс лексического анализатора 

Класс используется для разделения текста исходного кода на лексемы -\/ лексический анализ. Принимает на вход файл с текстом исходного кода (\char`\"{}text.\+txt\char`\"{}), написанного на языке DC++, на выходе выдает последовательность лексем с текстом, типом и строкой кода, в которой она написана (\char`\"{}separated.\+txt\char`\"{})
